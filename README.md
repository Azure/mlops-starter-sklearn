# MLOps with Azure Machine Learning
## 概要
本リポジトリは MLOps のプラクティスに沿って Azure Machine Learning と GitHub を利用して、モデル学習、デプロイメント、監視をする際のサンプルコードを提供します。

## 前提条件
- Dataset : Taxi
- Azure Machine Learning
    - Workspace
    - Compute
- GitHub
- Visual Studio Code
